About ListenUP 🎵

ListenUP is a simple, lightweight web music player built for streaming songs, creating playlists, and enjoying your favorite music directly in your browser.

🚀 Features

  • 🎵 Stream music online
  • 🔍 Search for songs and artists
  • ❤️ Create & manage playlists
  • 📱 Responsive design
  • 🔊 Volume control (coming soon)

⚙️ Tech Stack

Built with HTML, CSS, and JavaScript. Hosted on Netlify with source code available on GitHub.
